UHVibe
Type: School Project

Layout: Featured image, Boxed Content, Single Column

Host: Meteor

Languages: HTML, CSS, JavaScript

Framework: Semantic UI

Applications: Adobe Creative Suite, Java, CSS, Meteor, Intellij

Project Details: As a final project for my Software Engineering class, we were put into groups to create a fully functional Meteor application. At first, I had not known what to expect. I had never worked with a group in the setting of using CSS, HTML and Javascript. I enjoy working with others, but at the same time, I am a visionary. When it comes to designing and the final look and feel of a product, I want it to be at its best and to excel expectations.

This was not an issue at all. After I offered to create the base design, we were able to work on the technical coding as a team. I learned alot from this experience. As my team members Jun and Mai were great with fixing technical errors and had great ideas to contribute. From this project I learned to use the Model View Controller MVC design pattern. With the combined use of Intellij, GitHub Desktop, Command Prompt, Meteor and my web broswer, I was able to share responsiblities and keep an overview of all changes at the same time.

The simultaneous editing was confusing at first. But once we figured out how to utilize the Projects tab in GitHub, which allows us to assign tasks to each group member we were off to a great start. Another key factor in group editing and design is to be descriptive of each change that is made. When you are commiting a change, large or small it is important to label and describe what the change is for. This way if another team member decides to merge or pull your changes they know exactly what it is for.

I found this project to be the most useful in enriching my knowledge of Software Engineering. Now that I have worked with my group for about a month or so, I am much more comfortable with using the various programs needed. I look forward to strengthening my skills in future ICS classes and professional endeavours.
